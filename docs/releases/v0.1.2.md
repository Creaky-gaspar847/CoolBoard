# CoolBoard v0.1.2

Safety patch release for sleep/wake fan control.

## Known Bugs Fixed

- Fixed a safety issue where closing the Mac could restore Auto briefly and then re-apply the previous manual fan preset during wake or darkwake.
- Invalidated pending manual fan writes when sleep or wake starts, preventing stale write tasks from completing after Auto restore.
- Updated AppleSMC restore bookkeeping so the app remembers Auto intent even when helper restore falls back to direct SMC or reports a recoverable failure.
- Added a regression test that verifies wake does not send a second manual fan write.

## Behavior Change

- Manual presets are not resumed after sleep or wake. After opening the Mac, apply a preset again if manual fan control is still needed.
